This is a sample project for CruiseControl that 
shows you can you compile Flex applications
and run unit tests against them.

These files should merged into the root of your CruiseControl 
installation.

To get these flex compiling examples working you will
need to add the a flex sdk to the root of your
Cruisecontrol folder.  You will also need the 
Flex Ant tasks and Flex Unit lib available from labs.adobe.com.
